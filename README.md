# crud-blog

### Routes
1. POST: /api/v1/sign-up
2. POST: /api/v1/sign-in
3. GET: /api/v1/sign-out
4. GET: /api/v1/all-user

5. POST: /api/v1/blog/create
6. GET: /api/v1/blogs
7. PUT: /api/v1/blog/update/:blogId
8. DELETE: /api/v1/blog/delete/blogId
